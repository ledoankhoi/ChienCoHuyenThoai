## Mô tả
Hướng dẫn sử dụng HTML Canvas API để tạo đồ họa, animation, text rendering với các fonts đi kèm.

## Cách dùng
- Load skill khi cần tạo canvas graphic, banner, hoặc animation
- Font files có sẵn tại `anthropic-skills/skills/canvas-design/canvas-fonts/`

## Ví dụ
- Vẽ shapes, gradients, patterns
- Text rendering với custom fonts
- Animation loop với requestAnimationFrame
