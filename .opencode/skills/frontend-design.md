## Mô tả
Hướng dẫn thiết kế frontend: layout responsive, micro-interaction, typography, color system, accessible components.

## Cách dùng
- Load skill: gọi `skill` tool với name `frontend-design`
- Sau đó yêu cầu thiết kế component/page cụ thể

## Nguyên tắc
- Mobile-first responsive
- CSS custom properties cho color/typography
- Smooth transitions, micro-interactions
- Semantic HTML, ARIA labels
- Không dùng framework CSS nặng — ưu tiên Tailwind utility classes
